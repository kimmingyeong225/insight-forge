# Insight Forge — Skills 번들: `insight-forge-crypto`

본 번들은 Insight Forge의 **암호화폐 분석 전용 확장 번들**이다. 주식 도메인의 `insight-forge-default`와 **동일한 코어 시스템**에서 작동하지만, 모든 분석 규칙이 암호화폐 시장 특성에 맞춰 재정의되어 있다.

본 번들의 존재 자체가 Insight Forge의 핵심 가치 — **"동일한 코어, 다른 규칙"** — 를 실물로 증명한다.

---

## 1. 번들 구성

`insight-forge-default`와 완전히 동일한 5파일 구조를 따른다.

| 파일 | 역할 | 주식 번들 대비 차이점 |
|------|------|---------------------|
| `data-rules.md` | 정규화 규칙 | 24/7 시간대 처리, 온체인 데이터 스키마 |
| `metric-rules.md` | 지표 계산 | 변동성 √8760 연율화, NVT 비율 등 4종 자체 지표 |
| `viz-rules.md` | 시각화 선택 | 코인별 고정 색상, 캔들스틱 자동 전환 |
| `insight-rules.md` | 인사이트 생성 | 11개 암호화폐 특화 규칙 (NVT, 활성 주소 등) |
| `report-rules.md` | 리포트 흐름 | 실시간 갱신 정책, DeFi/트레이더 시나리오 |

## 2. 주식 번들과의 핵심 차이점

| 항목 | `insight-forge-default` | `insight-forge-crypto` |
|------|------------------------|------------------------|
| 시간 단위 | 일별 (252일 영업일) | 시간별 (8760시간) |
| 변동성 임계값 | 25% 이상 = 높음 | 100% 이상 = 높음 |
| MDD 임계값 | -25% 이상 위험 | -50% 이상 위험 |
| 펀더멘털 지표 | PER, PBR (재무제표) | NVT, 활성 주소 (온체인) |
| 기본 통화 | KRW | USD/KRW |
| 갱신 주기 | 일 단위 | 분/초 단위 |
| 독자적 지표 | 분산투자 점수, 집중도, 건강도, RAR, 모멘텀 | NVT 비율, 활성 주소 모멘텀, 변동성 조정 위험, 펀더멘털 괴리도 |

## 3. 도메인 이식의 실증

본 번들은 다음을 증명한다.

### 3.1 코어 시스템 무수정 원칙
주식 번들 → 암호화폐 번들 전환 시 **시스템 코어(`src/core/*`) 코드는 한 줄도 수정되지 않는다.** 변경되는 것은 오직 5개의 .md 파일뿐이다.

### 3.2 동일한 파이프라인
6단계 파이프라인(데이터 입력 → 정규화 → 지표 계산 → 시각화 선택 → 인사이트 생성 → 리포트 구성)이 그대로 작동한다.

### 3.3 재사용 가능한 구조
본 번들의 구조를 그대로 따라 다음 번들들을 작성할 수 있다.
- `insight-forge-esg` (ESG 분석)
- `insight-forge-bond` (채권 분석)
- `insight-forge-fund` (펀드 평가)
- `insight-forge-realestate` (부동산 분석)

## 4. 사용 시나리오

### 4.1 일반 암호화폐 투자자
시나리오 = `crypto_default` 적용. BTC·ETH·알트 보유 비중을 분석하여 변동성·NVT·활성 주소 모멘텀 기반 종합 진단 제공.

### 4.2 DeFi 투자자
시나리오 = `defi_focused` 적용. 온체인 분석 섹션이 강조되며 TVL, 스테이킹 비율 등이 추가 표시된다.

### 4.3 단기 트레이더
시나리오 = `trader_realtime` 적용. 실시간 KPI가 10초마다 갱신되고 캔들스틱 차트가 기본 표시된다.
